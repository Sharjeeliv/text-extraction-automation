About

The Text Extraction Automation (TEA), is designed for providing an scalable and automated approach to large-scale text analysis, particularly in downstream keyword analysis of sections of interest. The program is fully open source and available on GitHub where the user can adjust the program to their specific needs and experiment with the various parameters. The current release is a beta version and lacks negative reinforcement, i.e., it can not identify the absence of a section of interest.

Usage

The web-interface of the program is greatly simplified. The user is required to provide the following information:

  • Texts Folder: A folder containing text files
  • Labels Folder: A folder containing label files
  • Label Word: A keyword to identify the label files
  • Extensions: A list of allowed extensions, separated by spaces
Presently, the only extensions supported are .txt files and sec-html, for the specific use case of extracting informations from sec financial texts. This will be expanded in a future release. Ensure that the label files are named with the keyword provided in the arguments exactly as it appears in the label word field. Moreover, it is assumed that each text file has a unique name and that the label files are named with the same name as the text files.
